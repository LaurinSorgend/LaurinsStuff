## How do I install these manifests?

Run the following:

```pwsh
scoop bucket add LaurinSorgend_LaurinsStuff https://github.com/LaurinSorgend/LaurinsStuff
scoop install LaurinSorgend_LaurinsStuff/<manifestname>
```
